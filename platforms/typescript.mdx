Typesync lets you generate TypeScript interfaces/types for the Firestore models defined in your Typesync schema.

## Platforms

- `ts:firebase-admin:12`: For backend projects that rely on the [Firebase Admin Node.js SDK (v12)](https://www.npmjs.com/package/firebase-admin).
- `ts:firebase-admin:11`: For backend projects that rely on the [Firebase Admin Node.js SDK (v11)](https://www.npmjs.com/package/firebase-admin).
- `ts:firebase:10`: For frontend projects that rely on the [Firebase Javascript SDK (v10)](https://www.npmjs.com/package/firebase).
- `ts:firebase:9`: For frontend projects that rely on the [Firebase Javascript SDK (v9)](https://www.npmjs.com/package/firebase).
